<aside>
💡 substring() → string객체의 시작 인덱스로부터 종료 인덱스 전까지 문자열의 부분 문자열을 반환

</aside>

```jsx
const str = "Mozilla";

console.log(str.substring(1, 3));
// Expected output: "oz"

console.log(str.substring(2));
// Expected output: "zilla"
```
